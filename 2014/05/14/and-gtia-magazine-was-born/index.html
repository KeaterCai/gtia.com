And GTIA Magazine was born

Date Wed 14 May 2014 Tags start

I first registered the domain gtia.com with the dream of creating a magazine for Atari users in 1999. Now, 15 years later, I'm ready to start!

Using some Open Source tools, it has become very easy to digitally publish. GTIA Magazine uses Github to host & collaborate and Pandoc to create the actual ePub file. The articles are written in a text format called Markdown which not only makes it easy to convert to different formats, it should make it easy to write viewers for any system.

Check out the site at Github!

Suggestions and comments can be made in this thread at AtariAge.
